# Leave Management Application

## Overview

The Leave Management Application is a Salesforce-based workflow automation solution developed using Apex and Lightning Web Components (LWC). The application allows users to submit, manage, and track leave requests through dynamic UI interactions and automated business logic.

This project focused on workflow-based automation, validations, record management, and Apex-LWC communication in Salesforce.

---

## Features

* Leave request submission and tracking
* Workflow-based automation
* Validation handling
* Dynamic UI behavior using LWC
* Apex controller integration
* Record management functionalities
* Business logic automation

---

## Technologies Used

* Salesforce
* Apex
* Lightning Web Components (LWC)
* SOQL
* JavaScript
* HTML
* CSS
* Validation Rules
